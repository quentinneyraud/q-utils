# q-dom

## Install

```bash
yarn add @qneyraud/q-dom
```

## Usage

```js
import { ... } from '@qneyraud/q-dom'
```

## Development

```
# clone repo
git clone git@github.com:quentinneyraud/q-dom.git
cd q-dom

# install dependencies
yarn

# Run example
npm run dev
```

## Build

```bash
npm run build
```
